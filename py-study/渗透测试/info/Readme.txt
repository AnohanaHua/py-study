用途：
	用于自动化信息收集，调用了teemo、subdomainsbrute、sublist3r、fierce、theharvester，后续还会不断的增加各类工具


用法：
	在/root/info/subdomain 目录下下载

	https://github.com/lijiejie/subDomainsBrute

	https://github.com/bit4woo/teemo

	https://github.com/aboul3la/Sublist3r

	fierce kali自带


       在/root/info/mail 下下载
       https://github.com/laramies/theHarvester





	在root目录下新建project/info


执行 python infogo.py -d xxx.com
执行完成后生成
domain_all.txt  fierce.txt  ipc_all.txt          sublist3r.txt  theharvester.txt
email_all.txt   ip_all.txt  subDomainsBrute.txt  teemo.txt

最主要的就是
domain_all.txt 存放所有的域名
domain_all.txt 存在所有的ip
email_all.txt 存放所有的邮箱
ipc_all.txt    存放所有的ip C段
