
写这个脚本的目的呢，就是帮单位排查有没有代码泄露事件
主要排查网站为：

github

码云

thinksaas

oschina

svnchina

codeproject

一共6个网站外加一个其他人写的工具gitprey

根目录下的keyword.txt 用来存放要搜索的关键字，支持中英文

执行后，会在根目录下生产code_dis.txt 里面存放有找到的可能的代码泄露
